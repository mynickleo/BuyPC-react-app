Сайт BuyPC - в работе использовался React, TanStack Query (React Query ver 4), Axios
Деплой проекта был осуществлен на сервер, т.к. Vercel, Netlify у меня не работали. А GitHub pages не позволяет сделать redirect по "псевдо" страницам

Итак, краткий функционал:
1. На сайте возможно осуществить вход, чтобы совершать покупки и увидеть приветствие на странице /more_info
    Логин: Admin (либо Админ)
    Пароль: admin

2. На сайте есть каталог со следующими ссылками:
    1. Процессоры - страница с процессорами (возможны покупки)
    2. Видеокарты - страница с видеокартами (возможны покупки)
    3. Оперативная память - страница с оперативкой (возможны покупки)
    4. Наши дела - страница с демонстрацией запросов

P.S. Пожалуйста, не обращайте внимание на цены. Купить железо по ним сейчас скорее всего невозможно :D

"Наши дела" - принимает данные с сервера https://jsonplaceholder.typicode.com/

Разбивка страницы сделана следующим образом:
1. Header с каталогом выполнен в index.html
    ***Каталог имеет стандартные ссылки a (не Link), но переход по ним выполнен в стиле React. Т.е. нет других html файлов, кроме index
2. Контент страницы представлен с помощью React

Проверка на авторизацию (доступ к контенту на странице) сделан с помощью HOC


Деплой был произведен на сервер и работает с помощью nginx. Сертификации по https нет, для этого нужен буквенный домен
